# Contributing to Awesome AI Tools for Entrepreneurs

Thank you for your interest in contributing to this curated list! We appreciate your help in keeping this resource up-to-date with the best AI tools for entrepreneurs, founders, and indie hackers.

## Submission Process

To add a tool to this list, please follow these steps:

### Step 1: Submit Your Tool to FutureStack
First, submit your tool to **[FutureStack](https://futurestack.com)** — our platform for discovering and reviewing AI tools.

- Go to [FutureStack](https://futurestack.com)
- Add your tool to the platform
- Provide detailed information about what your tool does, pricing, features, and use cases

### Step 2: Wait for Approval
The repository maintainer will review your submission on FutureStack and approve it. This ensures all tools in this list have been vetted and meet our quality standards.

### Step 3: Get Your FutureStack Tool Page Link
Once approved, your tool will have its own page on FutureStack. Copy the link to your tool's FutureStack page (e.g., `https://futurestack.com/tools/your-tool-name`).

### Step 4: Create a Pull Request
Create a pull request to this repository with the following:

- **Title**: Use the format `feat: add [Tool Name] to [Category] section`
- **Description**: Include:
  - A brief description of what the tool does
  - The link to your tool's page on FutureStack
  - Which category it belongs in

**Example PR:**
```
## Summary

Adds [MyTool](https://futurestack.com/tools/mytool) to the **Marketing and Growth** section.

MyTool is an AI-powered platform that helps entrepreneurs... [description]

## Entry Added

**[MyTool](https://futurestack.com/tools/mytool)** — Paid  
Brief description of the tool and what it does.
```

### Step 5: Merge
Once your PR is reviewed and approved, it will be merged into the repository.

## Important Guidelines

⚠️ **All tools in this list must link to their FutureStack page, not directly to the tool's website.** This ensures:
- The list maintains a single source of truth
- All tools are vetted before inclusion
- The list auto-updates based on approved tools on FutureStack

❌ **Direct links to your tool's website will not be accepted.** Please follow the FutureStack submission process first.

## Categories

Tools are organized into the following categories:
- Content Creation
- Marketing and Growth
- Finance and Analytics
- Customer Service
- Design and Video
- Productivity and Automation
- Development and Code
- And more...

If you're unsure which category your tool belongs in, feel free to ask in your PR!

## Questions?

If you have any questions about the submission process, please open an issue or leave a comment on this repository.

Thank you for contributing! 🚀
